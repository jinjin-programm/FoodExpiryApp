<action>
Update or create `MnnYoloPostprocessorTest.kt` with comprehensive tests for ALL pure functions:

1. **parseDetections tests:**
   - Normal case: valid float array with multiple detections → correct DetectionResult list
   - Empty array → empty list
   - Single detection → list with one item
   - Detections with zero confidence → filtered or included (verify behavior)

2. **applyNms tests:**
   - No overlapping boxes → all returned
   - Fully overlapping boxes (IoU > threshold) → only highest confidence kept
   - Partially overlapping boxes → appropriate suppression
   - Empty input → empty output
   - Single box → single box returned

3. **cropDetection tests:**
   - Normal crop within bounds → valid Bitmap crop
   - Box extends beyond image edges → clamped to image bounds
   - Zero-area box (x1==x2 or y1==y2) → null or empty bitmap
   - Negative coordinates → clamped to 0

4. **mapLabelToCategory tests:**
   - Known food labels → correct FoodCategory
   - Unknown label → default category (OTHER or similar)
   - Null/empty label → default category

Use JUnit 4 or 5 (check existing test patterns in the project). Each test should have a descriptive name.
</action>
