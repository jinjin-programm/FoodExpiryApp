<action>
Create `app/src/main/java/com/example/foodexpiryapp/inference/tflite/TFLiteYoloEngine.kt`:

Class `TFLiteYoloEngine` implementing `YoloDetector`, annotated `@Singleton`.

Constructor injects:
- `@ApplicationContext context: Context`
- Config values: model asset path (default "yolov8n.tflite"), input size (640), confidence threshold (0.5f), max detections (8)

loadModel() implementation:
1. Load model ByteBuffer from `context.assets.open(modelPath).use { it.readBytes() }` — map to ByteBuffer
2. Create `Interpreter.Options().apply { setNumThreads(4) }`
3. Initialize `Interpreter(modelByteBuffer, options)`
4. Set `isLoaded = true`
5. Wrap in try/catch, return false on failure
6. Must run on Dispatchers.IO

detect(bitmap) implementation:
1. Check isLoaded, return emptyList if not
2. Resize input bitmap to 640x640 using `Bitmap.createScaledBitmap`
3. Convert to ByteBuffer: allocateDirect(4 * 1 * 640 * 640 * 3), order ByteOrder.nativeOrder()
4. Iterate pixels, extract R/G/B as float normalized to [0, 1]
5. Create output Map: output 0 → Array(1) { Array(maxDetections) { FloatArray(4) } } (boxes), output 1 → Array(1) { FloatArray(maxDetections) } (scores), output 2 → Array(1) { FloatArray(maxDetections) } (classes)
6. Run `interpreter.runForMultipleInputsOutputs(arrayOf(inputBuffer), outputMap)`
7. Post-process: iterate detections where score > confidenceThreshold
8. Map boxes from [0,1] normalized to original bitmap coordinates
9. Create DetectionResult for each: boundingBox=Rect(x1.toInt(), y1.toInt(), x2.toInt(), y2.toInt()), confidence=score, classId=classId.toInt(), label="", category=FoodCategory.OTHER
10. Return list capped at maxDetections

unloadModel() implementation:
1. Close interpreter if not null
2. Set interpreter = null, isLoaded = false
3. Must run on Dispatchers.IO

NOTE: The exact output tensor format depends on the exported model. If NMS is included in the model export (recommended per research), there will be 3 output tensors: boxes, scores, classes. If NMS is NOT included, there will be 1 output tensor of shape [1, 84, 8400] requiring custom NMS post-processing. The implementation should handle the NMS-included case first and add fallback for non-NMS.
</action>
