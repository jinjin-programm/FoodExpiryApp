<objective>
Create a safety net before v2.0 changes: tag the current stable state and update .gitignore for new artifact types.

Purpose: Every risky migration needs a known-good rollback point. Tagging v1.1.0-backup means the team can `git checkout v1.1.0-backup` at any time to restore the shipped v1.0 state. Updating .gitignore prevents new MNN model files, native libs, and build artifacts from being accidentally committed.

Output: Git tag `v1.1.0-backup` pointing to current HEAD, updated .gitignore with v2.0 entries.
</objective>

<task type="auto">
  <name>Task 1: Create v1.1.0-backup git tag</name>
  <files>.gitignore</files>
  <read_first>
    - .gitignore (current ignore rules)
  </read_first>
  <action>
    1. Stage and commit any outstanding changes (check `git status` first). If working tree is dirty, commit with message "chore: pre-backup commit before v2.0 migration".

    2. Create an annotated git tag pointing to the current HEAD:
    ```
    git tag -a v1.1.0-backup -m "Backup tag before v2.0 AI Vision Engine Overhaul migration. Stable v1.0 + v1.1.0 UX state with 11 plans shipped, 14/14 requirements validated."
    ```

    3. Verify the tag exists and points to the correct commit:
    ```
    git tag -l v1.1.0-backup
    git log -1 --oneline v1.1.0-backup
    ```

    4. Update .gitignore to add v2.0 artifact patterns. APPEND the following block at the end of the existing .gitignore file (do NOT remove any existing entries):

    ```
    # v2.0 — MNN inference engine
    *.mnn
    *.mnn.weight
    *.mtok
    app/libs/MNN-*.aar

    # v2.0 — Model download artifacts
    *.part
    *.meta
    llm/

    # v2.0 — Native build intermediates
    .cxx/
    app/.cxx/
    ```

    5. Stage and commit the .gitignore update:
    ```
    git add .gitignore
    git commit -m "chore: update .gitignore for v2.0 MNN and model download artifacts"
    ```

    Do NOT push the tag or commit. The tag and commit stay local until the user explicitly pushes.
  </action>
  <verify>
    <automated>git tag -l v1.1.0-backup && git log -1 --oneline v1.1.0-backup && grep -c "mnn" .gitignore</automated>
  </verify>
  <done>

    - .gitignore contains entries for `*.mnn`, `*.mnn.weight`, `*.mtok`, `*.part`, `*.meta`, `MNN-*.aar`, `.cxx/`
    - Both changes are committed locally
  </done>
